# Screenshot Checklist

Use this page to capture the final screenshots for your report or presentation. The sections below match the polished screens in the current project.

## Recommended Screenshots

1. Login screen
   Show the polished landing layout and demo credentials card.

2. Dashboard overview
   Show summary cards, recent services, and Phase 3 feature highlights.

3. Services page
   Show the create-service form and the search, filter, and sort toolbar.

4. Metrics page
   Show live cards plus the CPU history chart.

5. Forecast page
   Show forecast cards, risk level, and recommendations.

6. Settings page
   Show dark-mode toggle and preference controls.

7. Mobile responsive view
   Capture one screen in a narrow viewport to demonstrate responsiveness.
